# Chatter App

A real-time chat application built using React.js, Node.js, and SQLite.
The application allows users to communicate through chat rooms and private messaging with a clean and responsive interface.

## Features

User Login Authentication
Dashboard/Home Page
List of Available Chat Rooms
Private Chats Between Users
Real-Time Chat Interface
Sidebar Navigation
Responsive UI
Logout Functionality

## Tech Stack
Frontend
React.js
React Router
CSS / Tailwind CSS
Backend
Node.js
Express.js
Database
SQLite

## Application Pages

### Login Page
Users can log in using their credentials.
Authentication handled through backend APIs.

### Home/Dashboard
Displays:
Available Rooms
Private Chats
Users can select a room or private conversation.

### Room Chat Page
Shows detailed messages of the selected room.
Allows sending and receiving messages.

### Private Chat Page
One-to-one conversation interface.
Displays detailed personal chats.

### Sidebar Navigation

Contains:

Home/Dashboard
Rooms
Chats
Logout Button
